# 35 Years of NRL, A Visual Story

A data-visualisation essay on 35 years of Australia's National Rugby League (1990–2025), covering venues, evolution, and teams.

**Live site:** https://cranberryjams.github.io/VIS-2/

By James Deng, for FIT2179 Data Visualisation, Monash University, Semester 1 2026.

## About

Twelve Vega-Lite charts arranged as a long-form essay in three parts:

1. **The Game and Its Grounds** — venue map and club lifespans, 1990–2025.
2. **How the Game Has Changed** — points per match, home vs away scoring, scoreline distributions, and the broadcast schedule.
3. **Teams and Their Reign** — total wins, interstate travel, ladder trajectories, head-to-head matchups, and club fingerprints.

## Project structure

```
index.html                Essay page; embeds all 12 charts via vega-embed
specs/                    Standalone Vega-Lite JSON specs, one per chart
datasets/                 CSV source data referenced by the specs
sketch.pdf                Early design sketches
revised/                  Revisions and working drafts
```

## Tech

- [Vega-Lite](https://vega.github.io/vega-lite/) 5.16 for all charts
- [Pure.css](https://purecss.io/) for layout
- Typography: Bebas Neue, Playfair Display, Inter (Google Fonts)
- Hosted on GitHub Pages

## Data sources

- **Match results** (`datasets/match_data.csv`) — NRL fixtures 1998–2025, from [uselessnrlstats](https://github.com/uselessnrlstats/uselessnrlstats/tree/main/data/nrl).
- **NRL results with venue and region** (`datasets/nrldata.csv`) — supplementary fixture data, from [thecruncherau / NRL Results](https://www.kaggle.com/datasets/thecruncherau/nrl-results) on Kaggle.
- **Club timeline and metadata** (`datasets/teams.csv`) — founding year, home city, premierships, brand colour. Compiled from each club's [Wikipedia](https://en.wikipedia.org/wiki/National_Rugby_League) article.
- **World coastlines** — [vega-datasets / world-110m](https://github.com/vega/vega-datasets).

## Running locally

The page loads spec and CSV files via `fetch`, so it needs a static server, not `file://`:

```bash
python -m http.server 8000
# then open http://localhost:8000/
```
